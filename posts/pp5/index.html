Spotify Tracklist Analysis - Personal Project

Objective

With the advancements within artificial intelligence, it is to no surprise that I wanted to take a shot at building my own machine Learning algorithms. Over the past year, I have been studying core machine learning concepts. This digit recognition model is my first project trying to converge machine learning models with data. The model takes input of how a user draws a digit and then outputs what the model believes what digit was drawn. I can see a bunch of cases where this can be utilized for accessibility features or transcribing written documents. This project further solidified that fact that I would love to become a data scientist one day. I am very optimistic for what the future holds! (PYTHON CODE AND OTHER DOCUMENTS WILL BE POSTED SOON)

Software used: PyCharm, ScikitLearn, Keras, TensorFlow
